Som entreprenør

NORMIA brukes til å etablere tydelig prosjekteringsgrunnlag før oppstart, styre utførelse med objekter/soner/kontrollpunkter, og levere komplett og etterprøvbart grunnlag ved overtakelse.

Som byggherre

NORMIA gir oversikt over hva som er prosjektert og levert, og sikrer beslutningsgrunnlag som tåler ekstern gjennomgang.

Som rådgiver / prosjekterende

NORMIA strukturerer prosjekteringen slik at den kan utføres og kontrolleres uten tolkning.

Som kontrollforetak

NORMIA gir entydig kontrollgrunnlag, sporbar verifikasjon og revisjonssikker dokumentasjon.

Som FDV- og driftsgrunnlag

NORMIA sikrer sammenheng mellom fysisk løsning og dokumentasjon, og håndterer endringer i drift som revisjoner.
